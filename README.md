# Morpion-Python
Voici mon tout premier projet en Python, un jeu de Morpion simple.
L'ordinateur commence la partie et joue toujours dansu ne case libre.

Le programme détecte automatiquement :

les victoires du joueur (O)
les victoires de l’ordinateur (X)
les matchs nuls
les cases invalides ou déjà occupées

Ce projet a été réalisé pour apprendre la gestion des fonctions, de la logique de jeu, et la manipulation des listes en Python.


🎮 Fonctionnalités

- Affichage graphique simple du plateau
- Validation totale de l’entrée utilisateur
- IA basique (choix aléatoire parmi les cases libres)
-Détection :
  - victoire horizontale
  - verticale
  - diagonale
  - match nul
